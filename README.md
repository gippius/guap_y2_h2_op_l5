# Лабораторная работа №5

Тема: "Обработка исключительных ситуаций".

В проекте реализован вариант 4: класс для работы с массивом вещественных чисел, вычисление минимального элемента, суммы между первым и последним положительными элементами, перестановка нулей в начало и обработка стандартных и пользовательского исключений.

## Сборка

```powershell
cmake -S . -B build -G "Visual Studio 17 2022"
cmake --build build --config Release
```

## Запуск

```powershell
.\build\Release\lab5_array_exceptions.exe
```
